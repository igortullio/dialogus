## Implementation Details

Reference TechSpec § Component Overview for the full primitive list. Tailwind v4's CSS-first tokens differ from v3's `tailwind.config.js` — use `@theme` directive in `globals.css` to declare tokens, then reference via `var(--*)`. shadcn primitives expect specific token names (`--background`, `--foreground`, `--primary`, etc.); align with shadcn's documented contract.

The smoke page is a development convenience; it lives at `/_smoke` (underscore prefix prevents accidental production routing per Next.js convention). Remove or gate behind `NODE_ENV` before closure.
